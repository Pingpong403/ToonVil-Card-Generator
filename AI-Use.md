# AI Usage

I plan to use AI to help me build this project. AI will not be making any design choices for me. Its only purpose here is to find answers that I would otherwise have to scour the Web for.

Here is a list documenting every time I have asked AI a question and how it impacted the final product:

- Help with starting a .NET project
- Help with using System.Drawing to generate a PNG
- Help with using Path to build a proper filepath for intermediary text images
